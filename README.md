# Note-Back
博客的后台，也可以当笔记本

## 数据库设计

```
artical（数据库）
|
|-------用户文章（表）
	    |
	    |------id
    	|------create_date
    	|------change_date
    	|------title
    	|------content
    	|------tags
```
```
note-user(数据库)
|
|-------用户
	|-----username
	|-----password
	|-----name
	|-----sex
	|-----phone
	|-----intro
```

## 状态编码
```
//注册编码
100 注册成功
101 已经被注册

//登录编码
200 登录成功
201 密码错误
202 用户不存在

//注销编码
300 注销成功
301 注销失败

//数据库编码
500 数据库错误
```
